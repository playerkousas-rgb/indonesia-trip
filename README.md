# 童遊世界‧印尼活動備忘｜v9.1 完整覆蓋版

## 這個壓縮檔是什麼？
這份是 **完整覆蓋版**，適合你直接：
- 覆蓋 Apps Script 的 `Code.gs`
- 覆蓋前端 `index.html`
- 覆蓋前端 `app.js`

## 緊急聯絡資料 v9.1 更新
### 個人緊急聯絡資料 (`emergency_member_info`)
現在會包括：
1. 本人電話
2. 自己的緊急聯絡人
3. 自己的緊急聯絡人電話
4. 領袖 / 香港支援電話
5. 印尼官方緊急電話：
   - 綜合緊急求助：112
   - 警察：110
   - 救護車：118 / 119
   - 消防：113
   - 搜救 BASARNAS：115
   - 天災協助：129
6. 酒店前台電話
   - 峇里酒店
   - 雅加達酒店

### 顯示邏輯
- 成員：只看自己 + 領袖 / 支援 + 印尼官方緊急電話 + 酒店電話
- 領袖 / 超管：可看全體成員，另外也附領袖 / 支援 / 印尼官方緊急電話 / 酒店電話

## 緊急聯絡清單
- 對應：`emergency_contacts`
- 顯示領袖及支援聯絡電話

## 緊急時應如何處理
- 對應：`emergency_actions`
- 顯示緊急情況下應如何做

## 部署步驟
### Apps Script
1. 用這份 `Code.gs` 整份覆蓋
2. 儲存
3. 重新部署 Web App

### 前端
1. 覆蓋 `index.html`
2. 覆蓋 `app.js`
3. push 到 GitHub / Vercel
